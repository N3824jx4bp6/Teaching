## 判斷式
生活中會用到許多決定性以及變數，電腦也是，遇到不同的情況，需要下不同的決策，再將內容回傳到函數中顯示出不同的結果。
### 單向判斷式
### if()
其語法如下：
```python=
    if(判斷式)：
      決策內容
```
這個意義是當括號內的判斷式內容為True時，就會執行決策內容的敘述，反之，判斷式為False時則不執行。  
舉個例子
```python=
    if(time>10):
      print('Time out!')
```
上述區塊設定為時間超過60的時候，則顯示Time out!
```python=
    if(score>=60):
      print('pass!')
```
### 雙向判斷式
### if...else
其實通常if,else會同時出現，情況如同if一樣，然而當if的判斷式為False時，則實行else裡面的內容  
舉個例子
```python=
    if(score>=60):
      print('pass!')
    else:
```
